# نظام إدارة المخزن الأوفلاين

## المشاكل التي تم إصلاحها

### ✅ مشكلة تغيير المشروع
- **المشكلة**: كان البرنامج يغلق نهائياً عند تغيير المشروع حتى لو اختار المستخدم "لا" في رسالة التأكيد
- **الحل**: 
  - إضافة متغير `_changing_project` لتتبع حالة تغيير المشروع
  - تعديل `closeEvent()` لتجنب رسالة التأكيد أثناء تغيير المشروع
  - تحسين ترتيب إغلاق النوافذ

### ✅ إزالة البيانات التجريبية التلقائية
- تم إزالة زر "إنشاء بيانات تجريبية"
- تم منع إنشاء البيانات التجريبية تلقائياً
- المشاريع الجديدة تبدأ فارغة تماماً

## كيفية الاستخدام

### تغيير المشروع بشكل صحيح:
1. اضغط زر "تغيير المشروع"
2. اختر "نعم" في رسالة التأكيد
3. ستفتح نافذة اختيار المشاريع
4. اختر المشروع المطلوب
5. اضغط "فتح المشروع المحدد"
6. ✅ سيفتح المشروع الجديد بدون رسائل تأكيد إضافية

### التنبيهات:
- أزرار التنبيهات تعتمد على البيانات الحقيقية فقط
- لا توجد بيانات تجريبية تلقائية
- المخزون المنخفض: يظهر العناصر ≤ 10 وحدات
- انتهاء الصلاحية: يظهر العناصر التي تنتهي في ≤ 10 أيام

## الملفات المهمة
- `main.py`: نقطة البداية
- `src/main_window.py`: الواجهة الرئيسية المحسنة
- `start_fresh.bat`: ملف تشغيل سريع

## التشغيل
```bash
python main.py
```

أو استخدم:
```bash
start_fresh.bat
```
