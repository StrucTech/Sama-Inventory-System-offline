# نظام إدارة المخزون

تطبيق سطح مكتب بلغة Python لإدارة المخزون يتصل بجداول جوجل لتخزين ومزامنة البيانات.

## المميزات

- **عرض المخزون**: عرض جميع عناصر المخزون في جدول منظم
- **إضافة العناصر**: إضافة عناصر جديدة مع الاسم والكمية وسعر الوحدة
- **تعديل الكميات**: تحديث كمية العناصر الموجودة
- **حذف العناصر**: حذف العناصر من المخزون
- **المزامنة الفورية**: جميع التغييرات تتزامن مع جداول جوجل
- **الحسابات التلقائية**: يتم حساب القيم الإجمالية تلقائياً

## المتطلبات

- Python 3.7 أو أحدث
- بيانات اعتماد Google Sheets API
- اتصال بالإنترنت لمزامنة جداول جوجل

## التثبيت

1. **قم بتحميل أو استنساخ هذا المستودع**

2. **قم بتثبيت حزم Python المطلوبة:**
   ```bash
   pip install -r requirements.txt
   ```

3. **إعداد بيانات اعتماد Google Sheets API:**
   - اذهب إلى [Google Cloud Console](https://console.cloud.google.com/)
   - أنشئ مشروعاً جديداً أو اختر مشروعاً موجوداً
   - فعّل Google Sheets API
   - أنشئ حساب خدمة وحمّل ملف بيانات الاعتماد JSON
   - ضع ملف بيانات الاعتماد في `config/credentials.json`
   - شارك مستند جداول جوجل مع البريد الإلكتروني لحساب الخدمة

4. **إعداد التطبيق:**
   - عدّل `config/config.json` ليطابق اسم مستند جداول جوجل الخاص بك
   - تأكد من أن اسم ورقة العمل يطابق إعدادك

## الاستخدام

1. **تشغيل التطبيق:**
   ```bash
   python main_arabic.py
   ```

2. **استخدام الواجهة:**
   - **تحديث**: إعادة تحميل البيانات من جداول جوجل
   - **إضافة عنصر**: انقر لإضافة عنصر جديد للمخزون
   - **تعديل الكمية**: اختر عنصراً وانقر لتعديل كميته
   - **حذف العنصر**: اختر عنصراً وانقر لحذفه
   - **الإعدادات**: الوصول إلى خيارات الإعدادات

## إعداد جداول جوجل

يجب أن يحتوي مستند جداول جوجل على الأعمدة التالية:
- **A**: اسم العنصر
- **B**: الكمية
- **C**: سعر الوحدة
- **D**: القيمة الإجمالية (يتم حسابها تلقائياً)
- **E**: آخر تحديث (يتم تحديثها تلقائياً)

سيقوم التطبيق بإنشاء العناوين تلقائياً إذا لم تكن موجودة.

## الإعدادات

عدّل `config/config.json` للتخصيص:

        "width": 800,

## استكشاف الأخطاء وإصلاحها

**مشاكل الاتصال:**
- تحقق من صحة ملف `credentials.json` ووجوده في المكان الصحيح
- تأكد من أن حساب الخدمة لديه وصول إلى مستند جداول جوجل
- تحقق من اتصالك بالإنترنت

**أخطاء الاستيراد:**
- تأكد من تثبيت جميع الحزم المطلوبة: `pip install -r requirements.txt`
- تحقق من استخدام Python 3.7 أو أحدث

**أخطاء الصلاحيات:**
- شارك مستند جداول جوجل مع عنوان البريد الإلكتروني لحساب الخدمة
- امنح صلاحيات "محرر" لحساب الخدمة
